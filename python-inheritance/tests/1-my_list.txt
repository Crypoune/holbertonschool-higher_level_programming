=============================
The ``1-my_list`` module
=============================

Using ``MyList`` class that inherits from list.

Basics
======

``print_sorted`` prints the list in ascending order without modifying the original list.

::

	>>> MyList = __import__('1-my_list').MyList
	>>> my_list = MyList()
	>>> print(my_list)
	[]

Test with multiple integers
============================

::

	>>> my_list.append(1)
	>>> my_list.append(4)
	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(5)
	>>> print(my_list)
	[1, 4, 2, 3, 5]

Test print_sorted
=================

::

	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]

Original list unchanged
=======================

::

	>>> print(my_list)
	[1, 4, 2, 3, 5]

Empty list
==========

::

	>>> my_list2 = MyList()
	>>> my_list2.print_sorted()
	[]

Negative numbers
================

::

	>>> my_list3 = MyList()
	>>> my_list3.append(-1)
	>>> my_list3.append(-5)
	>>> my_list3.append(3)
	>>> my_list3.print_sorted()
	[-5, -1, 3]

One element
===========

::

	>>> my_list4 = MyList()
	>>> my_list4.append(42)
	>>> my_list4.print_sorted()
	[42]
